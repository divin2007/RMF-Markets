# Amabwiriza y'Umucuruzi (Seller Guide)

1. **Gufotora Igicuruzwa:** Buri gicuruzwa cyose kigomba kuba gifite ifoto kugirango kigaragare.
2. **Kwakira Komande:** Iyo ubonye komande, tegura ibintu byose vuba.
3. **QR Code:** Mbere yuko umutware atwara ibicuruzwa, asabwa kubifotora maze agasikana QR code iri ku nzu yawe (stall).
4. **Kwishyurwa:** Amafaranga yawe ayikubye 98.5% yoherezwa muri waledi (wallet) yawe ako kanya iyo ibicuruzwa bigeze ku mukiriya.
